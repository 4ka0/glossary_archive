# Glossary Archive

A simple webapp for archiving and searching previous translations and glossaries.

### Built using:

* Python 3.9
* Django 4.0.6
* django-crispy-forms 1.14.0
* translate-toolkit 3.7.0
* whitenoise 6.1.0
* environs 9.5.0
